Surge Engine

A retro game engine for making your dreams come true!

Make your dreams come true!

If you dream of making a retro game, you're good to go. It's easy to get started: download the engine and get creative! It includes:

  • A base game called Surge the Rabbit
  • A built-in level editor
  • Ready to use items, scripts, and more
  • Free to use game assets under Creative Commons licenses

Featuring Surge the Rabbit

Surge the Rabbit is a fun retro platformer bundled with the engine. You can play it and use it as a base to create your own games!

  • Play on amazing levels!
  • 360° physics system
  • Local multiplayer
  • Easy to MOD

A powerful engine for retro games

SurgeScript is easy and powerful scripting for games. Use it to create anything: items, bosses, gameplay mechanics, you name it!

  • Fun to play with - seriously!
  • Beginner-friendly
  • Rich set of features
  • Great tool for learning how to code

Open-source & cross-platform

Surge Engine is free & open source sofware. It's a lightweight engine for Desktop and mobile. Run your games on multiple platforms:

  • Microsoft Windows
  • GNU/Linux
  • Android
  • macOS (community builds)
  • ChromeOS, SteamOS & Raspberry Pi OS (via Flatpak)
