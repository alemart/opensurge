<b>Исполните ваши мечты!</b> Open Surge Engine - это 2D ретро игровой движок с открытым исходным кодом для создания игр и исполнения желаний!

<b>Это весело!</b> Surge the Rabbit - это включённая в комплект игра платформер, созданная с использованием Open Surge Engine. Она выполнена в духе классических 16-битных игр о Сонике из 1990х. Играйте за Surge в захватывающих уровнях наполненными испытаниями!

<b>Выпустите свою креативность!</b> Создавайте свои удивительные игры и играйте в них на ПК и мобильных устройствах! Делитесь вашими играми с друзьями! Это безграничное веселье!

<b>Мощный движок для ретро игр!</b> Один из ключевых элементов движка - SurgeScript, язык скриптинга для игр. Используйте его, чтобы создавать новые игровые механики, персонажей с особыми способностями, боссов и многое другое! Предел - только небо!

Open Surge Engine это отличный инструмент для изучения разработки игр, программирования, цифрового рисования и сущности свободного и открытого ПО в игровой форме.

На заметку: рекомендуется использовать ПК чтобы создавать свои игры. Вы можете играть в них на ПК и мобильных устройствах. Пользователям Android ТВ необходимо использовать геймпад чтобы играть и мышь чтобы использовать редактор уровней
